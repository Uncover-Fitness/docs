## About Uncover

Uncover is a unified training and client management platform that brings together trainers, nutritionists, and clients in one cohesive ecosystem — web dashboard, iOS app, and Android app.

We combine clinical-grade diagnostics, AI-driven coaching, and wearable integration to help individuals and organizations optimize human performance, recovery, and lasting health habits.
